# Mind Mate

> A Next.js/React-based mental‑health dashboard  
> Helps users track moods, notes, tasks and get media recommendations.

## 🚀 Features

- ✅ Mood tracker & logger  
- 📓 Notes manager  
- 📅 Tasks manager  
- 📊 Weekly dashboard  
- 🎵/📺 Media recommendations  
- 🔐 Authentication form  
- 🎨 Dark/light themes with responsive UI built on custom components

## 🛠 Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- React / TypeScript
- PNPM for package management
- Tailwind CSS (via PostCSS)
- Custom UI library under `components/ui`
- Hooks under `hooks/`
- Utility helpers in `lib/utils.ts`

## 🧩 Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/praveenap2310-art/MIND-MATE.git
   cd MIND-MATE
2.Install dependencies

# using PNPM (recommended)
pnpm install

# OR with npm
npm install

# OR with yarn
yarn install

3. Run the development server
pnpm dev

4.Build for production
pnpm build
pnpm start
